如何写出一篇hexo博文

Sep 30, 2017
文章目录
  1. 1. 第一步
  2. 2. 第二步
  3. 3. 总结

第一步

命令行中输入hexo new post 博客名称

这样就会在source目录下的_posts文件夹中新建一个.md文件。

第二步

  1. hexo clean 清除原有配置
  2. hexo generate 打包项目
  3. hexo deploy 发布博客

总结

由于hexo是静态博客，所以每次修改都需要上传一次。所以为了减少上传次数，建议上传前先使用hexo server命令在本地查看效果。
